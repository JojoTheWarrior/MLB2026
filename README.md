# Measurements for Sabermetrics
- [ ] WOBA (Weighted On-Base Average) measures hitter's overall offensive contribution by assigning values to walks, singles, doubles
- [ ] ERA (Earned Run Average) shows how many runs a **pitcher** allows per 9 innings.

# Rules
- [ ] Players get out through **strikeouts**, **flyouts** (caught in the air), **groundouts** (fielded and thrown to first), **forceouts** (fielder touches base before the runner) - happens with forced plays, when one runner pushes the next away.

# Postseason
- [ ] Playoff tournament held after the regular season, competing in **elimination** rounds (Wild Card, Division, Championship), and ending with the World Series.
